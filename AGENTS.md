# Agent Instructions

Rules for AI agents working on this project.

**Always follow these rules when making changes:**

1. **Study docs/PROJECT.md** - Before doing anything. Provides overview an documentation links
2. **Follow best practices** - Use industry-standard patterns and conventions
3. **Run tests** - `uv run pytest` after any code change
4. **Run pre-commit** - `uv run pre-commit run --all-files` before completing work
5. **Update documentation** - Keep docs in sync with code changes
6. **Update docs/learnings.md** - With discovered patterns for future iterations
7. **Code in `docs/specs/*` files are for reference/example** -  do not use it blindly.
8. **`docs/specs/*` files may be ambiguous or incomplete** - Always ask clarifying questions.
